# [Ogihara Ryo's Web site](http://ogihara-ryo.herokuapp.com)

## 開発環境
- Ruby 2.3.0
- Rails 4.2.5

## インストール
```
cp config/database.yml.example config/database.yml
bundle
rake db:create
rake db:migrate
```
